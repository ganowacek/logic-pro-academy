# Contributing

Thanks for helping make music production easier to learn.

## Before you begin

- Keep lessons practical, visual, and friendly to complete beginners.
- Teach production techniques without reproducing copyrighted songs, stems, lyrics, or recordings.
- Explain what a tool is, why it matters, how to use it in Logic Pro, and how it serves the learner's track.
- Preserve the dark, focused creative-tool aesthetic and test responsive behavior.

## Development workflow

1. Fork the repository and create a focused branch.
2. Install dependencies with `npm install`.
3. Run the app with `npm run dev`.
4. Validate with `npm run typecheck` and `npm run build`.
5. Open a pull request explaining the learner impact of the change.

Please keep pull requests small enough to review comfortably and avoid unrelated formatting changes.
